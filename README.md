Full-Stack LMS Platform (MERN + RTK Query + Stripe + Cloudinary + Shadcn UI)

A complete Learning Management System (LMS) built using the MERN stack, Redux Toolkit (RTK Query), Stripe payments, Cloudinary uploads, and Shadcn UI.
This platform allows users to browse courses, purchase them, track progress, and manage learning, while giving admins tools to create, edit, and manage course content.

Features
Student Features

View published courses

Advanced search and filtering

Course details page

Purchase courses using Stripe

Track lecture progress

Update and manage user profile

Light/Dark theme mode

Admin Features

Create, edit, publish, and unpublish courses

Upload videos and course materials using Cloudinary

Manage lectures (add, edit, delete)

Admin dashboard for course management

View purchased learners

Edit course information dynamically

Frontend

React with Vite

Redux Toolkit + RTK Query

Shadcn UI

Protected routes for student/admin

Responsive modern UI

Hero section, Navbar, Course pages, Dashboard

Backend

Node.js + Express

MongoDB + Mongoose

Authentication using JWT and Cookies

Stripe payments and webhook

Cloudinary + Multer for media uploads

RESTful API structure
